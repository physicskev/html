Chrome Extensions

A collection of useful tools I've developed

Gemini Chats Index

  • Outputs all your Gemini Chats links to a spreadsheet.
  • Helps access old chats buried in hundreds of links.

Web Scraper Pro

  • Scrapes any webpage as HTML, and downloads all links and images.
  • Useful to quickly download table of contents — or memes, lol.

Text Counter and Downloader

  • Type or paste text in window to count words and characters, & optionally download as .txt file.
  • Also a quick workaround for the annoying inability to create a new text file on macOS.

Grab cURL

  • Converts browser network requests to cURL commands.
  • Useful for debugging network interactions and API calls.

SHA-256 Hash Generator

  • Creates SHA-256 Hashes of any text entered or pasted.
  • Ideal for testing out various hashing and encryption protocols.
